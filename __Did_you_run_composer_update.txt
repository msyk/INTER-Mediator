Before you try the INTER-Mediator, you have to execute the below command.

cd INTER-Mediator  # Set the current directory to the root of INTER-Mediator.
composer install    # PHP and JavaScript libraries are going to be installed.

* For Windows Installation
Some libraries require creating symbolic links (shortcuts).
Run the command as an administrator or turn on Developer Mode before you proceed.

* Workaround in case of Failure to execute composer install/update
#1: Try to install node, and do them again.
or
#2: Try to execute composer and pnpm separately as following:
cd INTER-Mediator
composer update --no-scripts
pnpm ci
